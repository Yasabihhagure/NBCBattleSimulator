# MBBTester Web App 起動手順

`http://localhost:5173/` にアクセスできない場合、以下の手順で開発サーバーを手動で起動してください。

## 重提要件

- Node.js がインストールされていること
- ターミナル（PowerShell または コマンドプロンプト）が利用可能であること

## 手順

1. **ターミナルを開く**
   - VSCode の場合: メニューバーの `Terminal` > `New Terminal` を選択します。
   - Windows の場合: スタートメニューから `PowerShell` または `コマンドプロンプト` を開きます。

2. **プロジェクトフォルダに移動する**
   以下のコマンドをコピーして実行（Enterキー）してください。

   ```powershell
   cd d:\workspace\Unity\MBBTesterWeb
   ```

3. **依存関係のインストール（初回のみ、またはエラーが出る場合）**
   もし `npm run dev` でエラーが出る場合は、まず以下を実行してください。

   ```powershell
   npm install
   ```

4. エラーが出る場合ポリシーを変更する

   ```powershell
   Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```

5. **開発サーバーを起動する**
   以下のコマンドを実行します。

   ```powershell
   npm run dev
   ```

6. **ブラウザでアクセスする**
   コマンド実行後、ターミナルに以下のように表示されます。

   ```text
     VITE vX.X.X  ready in X ms

     ➜  Local:   http://localhost:5173/
   ```

   `http://localhost:5173/` をブラウザで開いてアプリを確認してください。

## サーバーの停止方法

サーバーを停止するには、ターミナルで `Ctrl + C` を押し、確認を求められたら `Y` を入力してください。
