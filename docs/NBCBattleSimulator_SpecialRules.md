
# 対象攻撃時の特別ルール

----

```planetext
[名称]:"亡霊への攻撃"
[効果]:対象者へのダメージは常に1となる
```

----

```planetext
[名称]:"狐の躱し"
[効果]:命中判定補正-2,攻撃がファンブルならランダムに選んだ同じチームの活動中の仲間一人に1D4ダメージを与える
```

----

```planetext
[名称]:"木製"
[効果]:"爆裂弾"が命中した場合ダメージが2倍になる
```

----

```planetext
[名称]:"飛翔体"
[効果]:射撃武器以外で攻撃する場合、対象の直前の攻撃が[攻撃2]である必要がある。
```
